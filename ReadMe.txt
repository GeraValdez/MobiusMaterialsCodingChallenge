To run this project, I recommend creating a virtual environment and installing 
all of the packages in the text file requirements.txt.

Open Terminal, and run Python -v to make sure you have installed Python 3.3 or above
Run pip install -r requirements.txt to install a copy of all of the libraries required 
to run this project in your own folder.
